Detailed Description

This is a minimal C++ interface to communicate with the TP-Link M7350 modem web gateway interface. This was tested to work with modem version 5. Author: Vincent Paeder License: GPL v3

◆ CustomDeleter

template<auto DeleterFunction>
using tplink::CustomDeleter = typedef std::integral_constant<std::decay_t<decltype(DeleterFunction)>, DeleterFunction>

Wrap a C deleter function for use with smart pointers.

This relies on the behaviour of the call operator that assumes that the left-hand side argument should be a function, and therefore gets cast to function pointer. At the same time, the integral_constant instance gives the wrapped value type as its own type.
